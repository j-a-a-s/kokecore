# Kaklen consumption gate

No library may migrate to Kaklen until every item below is completed for the
specific package and version.

- [ ] Licencia e IP aprobadas
- [ ] Stack compatible
- [ ] API pública estable
- [ ] Sin imports internos
- [ ] Unit tests
- [ ] Integration tests
- [ ] Consumer contract tests
- [ ] Cobertura mínima
- [ ] SAST
- [ ] Secret scan
- [ ] Dependency audit
- [ ] SBOM
- [ ] Tarball validado
- [ ] README humano
- [ ] CHANGELOG
- [ ] Changeset
- [ ] Rollback documentado

Evidence must identify the exact package version and commit. A local `link:`
dependency is not evidence of a distributable contract.
